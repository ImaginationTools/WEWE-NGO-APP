# WEWE-NGO-APP
## Widows & Orphans Database Management System (WODMS) or (BIMS)

### Project Tracks:
- **Mobile:** .NET MAUI (Field Data Capture)
- **Backend:** Azure Functions (Python/ODK Protocol)
- **Admin:** Teams Embedded App (Case Management)

### Setup:
1. Open in VS Code.
2. Install recommended extensions.
3. Check /docs for the XLSForm schema.
